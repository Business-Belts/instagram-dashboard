# BB Instagram Performance Dashboard

Een wekelijks analytics dashboard voor Instagram, gebouwd op Later CSV exports.

## Hoe gebruiken

1. Open Later (desktop) → Analytics
2. Exporteer vier CSVs:
   - **Stories** → `Detailed_Story_Performance_...csv`
   - **Reels** → `Detailed_Reel_Performance_...csv`
   - **Posts** → `Detailed_Post_Performance_...csv`
   - **Profiel** → `Profile_Growth_and_Discovery_...csv`
3. Open het dashboard (via GitHub Pages link)
4. Upload de vier bestanden → klik **Dashboard bouwen**

## GitHub Pages instellen

1. Push dit project naar een GitHub repository
2. Ga naar **Settings → Pages**
3. Kies **Deploy from branch → main → / (root)**
4. Je dashboard is live op: `https://[jouw-naam].github.io/[repo-naam]/`

## Wat het dashboard toont

- **Overzicht** — KPIs, follower beweging, bereik per format, top performers
- **Reels** — views timeline, volledige reel tabel
- **Stories** — completion rate trend, top stories
- **Posts** — bereik timeline, post detail
- **Groei** — follower gain/loss per week, cumulatieve trend
- **Inzichten** — automatisch gegenereerde strategische aanbevelingen

## Technisch

Puur client-side HTML — geen server nodig, geen account, geen cookies. Data blijft lokaal in je browser.
